# Gym Workout Planner & Progress Logger (Console)

**Language:** Java  
**Type:** Console application

## Features
- Add/Edit/Delete Exercises
- Create Workout Plans and add entries (sets/reps/weight)
- Log Workout Sessions by date (confirm planned values or enter actual values)
- View workout history
- View progress (Personal Record = max weight, Last Performed date)
- Save/Load using local serialization file (`workout_data.ser`)

## Compile & Run (Terminal)
```bash
cd src
javac *.java
java Main
```

## Operating Instructions
1. Choose **Manage Exercises** and add exercises first.
2. Choose **Manage Workout Plans** to create a plan and add entries.
3. Choose **Log Workout Session** to record a workout.
4. Use **View Workout History** and **View Exercise Progress** to see results.
5. Use **Save** to store data and **Load** to restore it.
